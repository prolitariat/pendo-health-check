--- EXTENSION NAME ---
Pendo Health Check

--- SHORT DESCRIPTION (132 chars max) ---
Instant Pendo installation grade with prioritized diagnostics, one-click issue export, inline debugger, and live service status.

--- DETAILED DESCRIPTION ---
Pendo Health Check gives you an instant letter grade for your Pendo installation. Open the popup on any page with Pendo installed and get a single, prioritized diagnostic report — no tab-switching, no guesswork.

REPORT TAB — Instant Installation Grade + Prioritized Diagnostics

Your Pendo installation is scored from A to F based on 11 runtime health checks and a deep setup analysis. Every issue is ranked by severity so you know exactly where to start.

Runtime Health Checks (11):
• Agent Detection — confirms pendo.js is loaded
• Initialization — verifies the agent initialized via pendo.isReady()
• Visitor ID — checks that a non-anonymous visitor ID is set
• Account ID — validates account-level identity
• Active Guides — reports loaded guides count and state
• Pendo Instances — detects dual installations and multiple script tags
• Agent Version — reports the running Pendo agent version
• API Key — validates the Pendo API key is present
• Data Host — checks CNAME configuration vs default CDN
• Network Requests — validates active network traffic to Pendo servers
• Feature Flags — detects non-default agent configuration flags

Setup Analysis (merged into the same view):
• Framework detection (React, Angular, Vue, Ember, Svelte, Next.js, Nuxt, jQuery)
• Snippet installation analysis (install method, placement, async loading)
• CSP header and meta tag compatibility check (directive-by-directive)
• CNAME-aware host detection for custom CDN and data endpoints
• CMP/GDPR consent detection (OneTrust, Cookiebot, Didomi, Osano, TrustArc, TCF v2.0)
• Visitor and account metadata validation (PII detection, data types, field naming)

Copy Issues to Clipboard — one-click priority-sorted diagnostic report with remediation. Paste into Slack, Jira, or a support ticket.

TOOLS TAB — Debugger + Developer Console
• Toggle Pendo Debugger — turns Pendo debugging on/off with one click
• Validate Install — runs pendo.validateInstall() with inline results (no DevTools needed)
• Validate Environment — runs pendo.validateEnvironment() with inline results

ICON BADGE
• Pendo-yellow badge shows total issue count at a glance
• Disable the badge entirely from Settings in the Tools tab
• No badge when everything passes

PENDO SERVICE STATUS
• Live status monitoring from status.pendo.io — only shown during outages or recent incidents
• Per-component health indicators (App, Data Pipeline, Guides, Integrations)
• Active incident banners with latest updates
• Service status included in copied diagnostic reports

FEATURES
• Installation grade (A–F) computed from all checks and setup analysis
• Every warn/fail check includes an actionable fix recommendation
• Smart network request messaging — diagnoses blocked requests without noise
• First-run onboarding tour walks you through every feature
• Feedback system with automatic PII scrubbing
• Version displayed in-app (syncs from manifest)
• Works on any page with Pendo installed
• No data leaves your browser — all checks run locally

Built for Pendo admins, implementation engineers, and customer success teams who need fast answers about agent health.
